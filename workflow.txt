[ 리포트 제작 플로우 ]

1. raw data 준비
   (pdf / docx / txt 등 리서치 원본)
        ↓
2. AI가 html 변환
   - 기존 디자인 시스템 적용
   - 파일 상단에 report-meta 블록 작성
   - 파일명 규칙 적용: {cat}_{slug}_{YYMMDD}_{author}.html
        ↓
3. reports/ 폴더에 파일 추가 후 main 브랜치 push
        ↓
4. GitHub Actions 자동 실행
   build.py — reports/ 스캔 → 메타 추출 → index.html 갱신 커밋
        ↓
5. Vercel 자동 배포


----------------------------------------------------------------


[ 카드뉴스 제작 플로우 ]

1. 완성된 리포트 html
        ↓
2. AI가 핵심 내용 추출 + 슬라이드 구성안 제안
        ↓
3. PPT 템플릿에 내용 입력
        ↓
4. 이미지 내보내기 → 인스타그램 게시
